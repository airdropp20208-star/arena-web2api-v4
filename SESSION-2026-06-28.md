### Cần làm tiếp:
1. **Reverse-engineer reCAPTCHA** — tìm cách bypass hoặc generate token server-side
2. **Hoặc dùng browser hoàn toàn** — abandon server HTTP, mọi request đi qua browser automation
3. **Hoặc tìm API khác** — Arena có thể có API nội bộ khác không cần reCAPTCHA
